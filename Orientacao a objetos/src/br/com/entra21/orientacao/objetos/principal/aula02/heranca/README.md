# Aprendendo Herança

## Herança é a abstração para aproveitar atributos e métodos de uma outra classe

> - Essa classe que está provendo atributos e métodos podemos chama-la de super classe
> - Para realizar essa herança em uma sub classe temos que usar a palavra reservada **extends**
> - Quando uma sub classe herda atributos e métodos da super classe não é mais necessário manter meus atributos e métodos escritos na minha sub classe
